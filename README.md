# Rice Price Movement Prediction in the Philippines (2019–2026)

**Subject:** ISY122 
**Project Title:** Temporal and Regional Determinants of Rice Price Movement in the Philippines: A Random Forest Classification Analysis of WFP Food Price Data (2019–2026)

---

## 📌 Overview

This project applies machine learning to predict the **directional movement** (up or down) of retail rice prices in the Philippines using historical data from the **World Food Programme (WFP)**. The goal is to understand how temporal factors (month, year), regional differences, and rice commodity types influence price changes.

The project is part of our final requirement for **ISY122**. We built and compared four classification models, with **Gradient Boosting** achieving the best accuracy.
